Welcome to the sentiment analysis chat API!

These are some urls you can use: :

See all users:

--> http://localhost:5000/users

Create an user and save it to the database:

--> http://localhost:5000/chat/create?participants=Name1,Namen&chat_name=chat

Add user to existing chat:

--> http://localhost:5000/chat/adduser?new_participant=Name&chat_name=chat

User adds message to chat room:

--> http://localhost:5000/chat/addmessage?message=text&to=chat&from=Name

Get messages from chat:

--> http://localhost:5000//chat/list?chat_name=Anexo

Analyze sentiments from chat:

--> http://localhost:5000//chat/sentiment?chat_name=Anexo/
